Perdidos & Achados

Sistema web para ajudar pessoas a divulgar e encontrar objetos perdidos ou achados.

Acessar aplicação online Ver código no GitHub

Sobre o projeto

Este projeto foi desenvolvido como parte do curso de Análise e Desenvolvimento de Sistemas, em formato de projeto de extensão. A plataforma conecta quem perdeu um item a quem o encontrou, por meio de anúncios organizados por categoria e local.

Funcionalidades

  • Cadastro e login de usuários
  • Publicação de anúncios de itens perdidos ou achados
  • Listagem pública dos anúncios mais recentes
  • Edição e exclusão apenas pelo dono do anúncio
  • Categorias: Documento, Eletrônico, Animal e Outro

Diferenciais técnicos

Serverless API em funções Node.js na Vercel
PostgreSQL Banco relacional na nuvem
Frontend leve HTML, CSS e JavaScript vanilla
Deploy automático Integração GitHub + Vercel

Stack tecnológica

  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Backend: Node.js — Vercel Serverless Functions
  • Banco de dados: PostgreSQL
  • Hospedagem da app: Vercel
  • Documentação: GitHub Pages + README

Como usar

  1. Acesse a aplicação pelo botão acima.
  2. Crie uma conta em Cadastrar-se ou faça login.
  3. Clique em Novo Anúncio para publicar um item.
  4. Na página inicial, visualize todos os anúncios cadastrados.
  5. Se o anúncio for seu, use os botões Editar ou Excluir.
